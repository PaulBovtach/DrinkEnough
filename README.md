# DrinkEnough
This is simple app that can estimate amount of water needed per day demand on gender, age and weight. The main functional of app is a counter of consumed cups of water. Counter refreshes every day, so you can track consume and stay higrated and healthy.


### Techologies
* **Programming language** [Swift]
* **Framework:** SwiftUI
